## 0.1.0

- Initial SDK foundation for MyFatoorah Flutter/Dart integrations.
- Added typed core configuration, result, logging, and exception models.
- Added internal Dio-based HTTP layer hidden behind SDK abstractions.
- Restricted Flutter payment API calls to merchant backend proxy mode only.
- Added Hosted Payment v3 create payment URL support.
- Added Hosted Payment URL launcher for Android, iOS, and Web.
- Added high-level checkout facade for Hosted Payment orchestration.
- Added Payment Details / Status lookup support.
- Added callback URL parser helpers.
- Added pure Dart webhook parser and signature verifier helpers.
- Added Embedded Payment session initiation support.
- Added README quickstart and security notes for the current SDK state.
